---
title: 'vue cli 配置开发环境下的sourcemap'
---   
### 配置方法：

vue.config.js:

```javascript
module.exports = {
  lintOnSave: false,
  devServer: {
    //开发环境下设置为编译好以后直接打开浏览器浏览
    open: true,
  },
  configureWebpack: (config) => {
    //调试JS
    config.devtool = "source-map"
  },
  css: {
    //查看CSS属于哪个css文件
    sourceMap: true,
  },
}
```
